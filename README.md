# JetStock - Capstone
JetStock is a full-stack Inventory and Product Manager application built for small business owners to manage products, categories, suppliers, inventory and stock movements in one place.

The application helps users maintain accurate inventory records, track stock-in and stock-out activity, monitor low-stock products, and organize product data through a clean dashboard interface.

---

## Project Purpose
Many Small businesses manage inventory manually through notebooks, spredsheets, and Whatsapp. which can lead to inaccurate stock counts, missing product details, and poor visibility into stock movement.

JetStock sovles this problem by providing a centralized web application for managing:
- Products
- Categories
- Suppliers
- Stock-in records
- Stock-out records
- inventory summaries
- low-stock reporting
